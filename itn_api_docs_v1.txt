info:
  title: InstantTrade Network API
  description: |
    ## Real-Time B2B Payment Infrastructure
    
    InstantTrade Network enables instant supplier payments while giving buyers payment terms
    through embedded working capital. This API provides endpoints for:
    
    - **Invoice Management** - Create, retrieve, and manage invoices
    - **Settlement Execution** - Instant atomic settlements with 3-leg transfers
    - **Pricing** - Transparent pricing quotes for payment terms
    - **Health Monitoring** - System health and audit endpoints
    
    ## Authentication
    All endpoints require API key authentication via `X-API-Key` header.
    
    ## Rate Limiting
    - Invoice creation: 100 requests/hour per supplier
    - Settlement execution: 1000 requests/hour per account
    - Read operations: 10000 requests/hour
    
    ## Invariant Enforcement
    All operations are subject to 30+ invariants that enforce:
    - Unique invoice IDs
    - Valid amounts ($100 - $10M)
    - Active accounts only
    - Settlement exactly once
    - Atomic transfers
    - Settlement within 5 seconds
    
    Failed invariants result in automatic rollback with detailed error messages.
